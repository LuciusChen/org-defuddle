Some CMSes leave unresolved template literals like the ones in this page's meta tags. Defuddle should skip over these and fall back to the next valid source.

In this fixture, og:title, og:site_name, og:description, and article:published_time are all template placeholders. The name meta description and the name author are both literal "..". The real values come from the schema and twitter:description tags.

** Introduction

Recurrent models typically factor computation along the symbol positions of the input and output sequences, as explored in prior work [fn:6] [fn:1] [fn:2]. Some models use a single citation [fn:5] while others reference two works [fn:3] [fn:4].

[fn:3] Jonas Gehring, Michael Auli, David Grangier, Denis Yarats, and Yann N. Dauphin. Convolutional sequence to sequence learning. 2017.

[fn:4] Alex Graves. Generating sequences with recurrent neural networks. 2013.

[fn:5] Sepp Hochreiter and Jurgen Schmidhuber. Long short-term memory. 1997.
